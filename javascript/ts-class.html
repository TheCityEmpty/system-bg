    # 学习 ts 四： ES6中的类和TypeScript中的类

    # ES5中的类是怎么实现的？

    在传统的面向对象编程语言（比如java,c++）中都有 “类” 的概念。在 ES5 中并没有相应的关键字来生成一个类。但是却可以通过构造函数和原型链来模拟一个“类” 的概念。将方法名的首字符设置为大写，则认为这个函数是个类。
